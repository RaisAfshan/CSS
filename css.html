Front End

The part of a website or application that users see and interact with, such as buttons, graphics, and text messages. Front end developers work on the visual and interactive aspects of a website or application, such as the layout, colors, and fonts. They also ensure that the application is compatible across devices and that users have an easy time navigating it

Back End

The part of a website or application that users cannot see, but that makes the front end possible. Back end developers work on the data and infrastructure that make the application work, such as the database, security, and site performance.
